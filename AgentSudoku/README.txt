Code édité via Visual Studio 2017

Les grilles de sudoku doivent être placées à la racine du dossier dans le fichier 'sudokus.txt' de la manière suivante :
-une grille par ligne
-les 81 valeurs se suivent sans espace (0 pour une valeur non attribuée)

Exemple de grille :

001700509573024106800501002700295018009400305652800007465080071000159004908007053
